# Prompt Engineering 

This file contains notes about Prompt Engineering.

## What is a Prompt? Composition of a formal prompt

A prompt in prompt engineering is a specific input or task given to a language model to generate a desired output.

We can break a prompt into 4 different components:

- Instructions
- Context
- Input Data
- Output Indicator

### Instructions

You can think of it as the heart of the prompt. The instruction tells the LLM model what task to perform.

### Context

Additional information to fine tune the instruction.

### Input Data

The data that the model will process.

### Output Indicator

It signals the AI model that we expect the answer now.
